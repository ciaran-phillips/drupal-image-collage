
-- SUMMARY --

This module takes a view with text and image fields, and displays it as a collage of images with the textual data
displayed on hover and click.

It's based on the Views Vertical Tabs module created by Jibran Ijaz.
All of the general structure of this project and interaction with the Views API has been created by him for the original module.

For a full description of the Views Vertical Tabs module, visit the project page:
  http://drupal.org/node/1906858
  
This module is not being maintained on an ongoing basis as a standalone drupal module,
it has only been created for use on the Giving What We Can website, so will not appear
on drupal.org, but does have a home on github at:
